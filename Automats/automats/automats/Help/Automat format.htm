Automat Files Format


This application can read standart comma-separated files (*.csv), and cvs extended with C++ style comments ("//").
File format described on two following examples.

Mura automat file example
Mura;
;d;+;-;.;e;|;
;0;1;;;
;S0;S1;Spt;Sptd;Se;Se+-;Se+-d;Sok;e;
;1;8;1;2;8;8;
;1;8;8;2;8;8;
;3;8;8;8;8;8;
;3;8;8;8;4;7;
;8;5;5;8;8;8;
;6;8;8;8;8;8;
;6;8;8;8;8;7;
;8;8;8;8;8;8;
;8;8;8;8;8;8;
;0;0;0;0;0;0;0;1;0;;;
;;;;;;
±ddd . dddd [e±dddd]; 		 First line contains one keywords, defining the type of automat: Mili, Mura, MM.
Line 2 difines set A, e.g. alphabet of incoming symbols, line 3 difines set Z (outgoing symbols alphabet), line 4 difines set S (states alphabet).

Next lines represents Transfers Table in which number of columns equal to power of set S, number of rows is equal to power of set A. Elements of the table are indexes of elements from set S (indexes assumed according to order of appearence elements in line 4).

Next lines contains the Outs Table, it's column number is equal to power of set S, it's row number depends on type of automat. See Automats Types for details. Elements of the table are indexes of elements from set Z

Any next lines are ignored and may contain any custom data.
Empty elements (";;") are ignored also in compatibility with MS Excel© reasons.

Accept		 First line still aytomat type keyword. MM responds stack memory automat.
Line 2 difines set A
Line 3 difines set Z
Line 4 difines set S
Line 5 defines set M (stack-symblos alphabet)



Next lines represents Managing Table(s): columns number is equal to power of set A, number of rows is equal to power of set M. Elements of the table are indexes of rules specifed below. There are so many tables as power of set S.


Next lines deines rules. See Stack-memory automats rules for rules specification. Rule format:
— keyword "Rule", then title of rule, separated from keyword by comma or space.
— <keyword>{space|comma}<parameter>

File parsing ends with last rule, so it must not contain trash text.
Empty elements (";;") are ignored.
